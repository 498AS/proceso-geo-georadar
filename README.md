## Contenido

- **8 fases** del proyecto: kick-off → cuestionario → personas → cata de prompts → auditoría → reunión preliminar → presentación → cierre → dashboard continuo.
- **Smart Brevity** en todo el cuerpo: lead bold por fase, bloques etiquetados (cómo / necesitamos / entregable / plazo).
- **Ejemplos clicables** ilustrativos: FIFA · Coca-Cola · Nike — uno por concepto clave (tesis, personas, cata, KPIs, plan de acción, dashboard).
- **Timeline**: 4 semanas en escenario óptimo, 6 semanas como máximo.
- **Compromisos del cliente** explícitos: interlocutor único, plazos de respuesta acotados (4 días para cuestionario y revisión post-presentación, 2 días para personas y cata).
